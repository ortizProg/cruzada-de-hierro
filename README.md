# Juego de Patrones

Proyecto inicial de Java con Maven para implementar un juego que utilizará patrones de diseño.

## Requisitos
- Java 17 o superior
- Maven 3.6 o superior

## Compilar el proyecto
```bash
mvn clean install
```

## Ejecutar el juego
```bash
mvn exec:java -Dexec.mainClass="com.juego.Main"
```

## Ejecutar pruebas
```bash
mvn test
```

## Estructura del Proyecto
- `src/main/java/com/juego/`: Código fuente principal.
- `src/test/java/com/juego/`: Pruebas unitarias.
